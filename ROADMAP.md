# LogEats ロードマップ

## 優先度の高い課題（直近の対応）
1. **チャット風 Web UI**
   - 現在のメイン記録画面（ダッシュボード）をチャット形式のインターフェースにリファクタリング。
   - ユーザーが写真やテキストを送信すると、AIが栄養素の情報とともに返信する。
   - 過去の記録（昨日など）も、チャットの履歴を上にスクロールするだけで確認可能。

## 将来の機能 / キラーコンテンツ
1. **外部チャットアプリ連携（Bot）**
   - **LINE連携（日本国内向け）:** ユーザーがLINE公式アカウントに食事の写真を送れるようにする。Botが画像を解析し、WebアプリのDB（Clerkのアカウント連携を利用）に記録を保存。即座にカロリーや栄養データをLINEで返信する。
   - **AIのキャラクター（ペルソナ）設定:** 毎日の記録を楽しく続けるために、AIのキャラクター設定（例：厳しいトレーナー、優しい管理栄養士、世話焼きなオカンなど）を用意する。
   - **Instagram DM / Discord / Telegram連携:** ターゲットユーザーに合わせてBotの展開先を広げる（例：カジュアルに写真を撮る層にはInstagram、筋トレ仲間のコミュニティ向けにはDiscordなど）。
